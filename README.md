# airline-traffic-analysis
Analysis of U.S. airline traffic data to identify peak and low travel seasons using Python 
## Dataset Description (توضیحات دیتاست)

The project utilizes the air traffic.csv dataset, which contains historical aviation traffic data. Below is the explanation of each column:

| Column Name | Description 
| :--- | :--- | :--- |
| Year | The year of the recorded data | سال ثبت داده |
| Month | The month of the recorded data (1-12) | ماه ثبت داده |
| Dom_Pax | Domestic Passengers volume | تعداد مسافران پروازهای داخلی |
| Int_Pax | International Passengers volume | تعداد مسافران پروازهای بین‌المللی |
| Pax | Total Passengers (Domestic + International) | کل مسافران (داخلی + بین‌المللی) |
| Dom_Flt | Number of Domestic Flights | تعداد پروازهای داخلی انجام شده |
| Int_Flt | Number of International Flights | تعداد پروازهای بین‌المللی انجام شده |
| Flt | Total Flights (Domestic + International) | کل پروازها (داخلی + بین‌المللی) |
| Dom_RPM | Revenue Passenger Miles (Domestic) | شاخص مسافر-مایل درآمدزایی در پروازهای داخلی |
| Int_RPM | Revenue Passenger Miles (International) | شاخص مسافر-مایل درآمدزایی در پروازهای بین‌المللی |
| RPM | Total Revenue Passenger Miles | کل شاخص مسافر-مایل درآمدزایی |
| Dom_ASM | Available Seat Miles (Domestic) | شاخص صندلی-مایل موجود در پروازهای داخلی |
| Int_ASM | Available Seat Miles (International) | شاخص صندلی-مایل موجود در پروازهای بین‌المللی |
| ASM | Total Available Seat Miles | کل شاخص صندلی-مایل موجود |
| Dom_LF | Load Factor for Domestic Flights (%) | ضریب بار (میزان پر بودن صندلی‌ها) در پروازهای داخلی |
| Int_LF | Load Factor for International Flights (%) | ضریب بار (میزان پر بودن صندلی‌ها) در پروازهای بین‌المللی |
| LF | Total Load Factor (%) | ضریب بار کل صندلی‌ها به درصد |
